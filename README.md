## What is CitizenBenefits?

A privacy-first AI agent that takes only anonymous situational facts (location, household size, income range, age, family circumstances) and estimates which major assistance programs a person is likely eligible for.

It explains the reasoning in plain language and provides direct links to official application portals.

Currently focused on US federal programs (SNAP, Medicaid/CHIP, EITC, LIHEAP, WIC) with plans for global expansion.

## Key Features

- 100% Anonymous - No names, emails, SSNs, or personal details ever collected
- Grounded in current official sources with live data parsers where possible
- Neutral, factual explanations in multiple languages
- Clear disclaimers on every estimate
- Modern, accessible web interface with strong security practices

## Tech Stack

- Google Agent Development Kit (ADK 2.0) Graph Workflow
- Gemini 3.5 Flash
- FastAPI + Cloud Run
- Agent Skills architecture (one SKILL.md per program)
- Rigorous security and evaluation framework
